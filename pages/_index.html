What is the VJET JavaScript Integrated Development Environment?

VJET JavaScript IDE is an Eclipse plugin that provides a fully integrated development environment for JavaScript - from authoring, to execution/test, and debugging. More

  • Code Assistance
  • Code Search
  • Syntax and Semantic Checking
  • Type Declarations using VJETDoc
  • Execution/Debugging

Why Would I Use VJET JavaScript IDE?

VJET JavaScript IDE replaces standard text editors or simple JavaScript editors used by many JavaScript developers today and brings common IDE features to JavaScript. VJET JavaScript helps developers focus on coding by providing assistance and support during the full development lifecycle.

  • Code JavaScript faster and accurately with code completion, code templates, and wizards
  • Search for JavaScript types and relationships using the call hierarchy, type hierarchy, references, implementers, etc.. Find code faster than grep.
  • Identify type errors using Java like semantic validation - are you using the JavaScript APIs or user defined type correctly? Find out before the user does.
  • Execute and debug JavaScript with full run-time and debug support.
  • Take advantage of a professional IDE for Eclipse with Java like tool features such as syntax highlighting, semantic highlighting, outline view, quick outline view, occurrence markers, and Eclipse editor features.
  • Supports Javascript 1.5 language constructs - conditionals, loops, literals, built in data types

Who Can Benefit From Using VJET JavaScript IDE

Anyone who authors JavaScript can benefit from using VJET JavaScript IDE, whether you build web pages or code JavaScript libraries. VJET JavaScript IDE supports native JavaScript, Browser, and DOM types for out-of-the-box support for most JavaScript usage. VJET JavaScript Type Libraries allow users to extend VJET JS support to 3rd party or custom JavaScript libraries. Users can use published VJET JavaScript type libraries or create their own. With VJET JS, you use only the features you need.
